# GDB
动态调试程序
安装
```py
1. wget http://ftp.gnu.org/gnu/gdb/gdb-8.0.1.tar.gz`
2. tar -zxvf gdb-8.0.1.tar.gz`
# 进入目录
3. cd gdb-8.0.1`
4. ./configure`
5. make
6. sudo make install
```
